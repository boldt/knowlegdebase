# PPA

## Add

```bash
sudo add-apt-repository ppa-url
```

## Remove

### Only repo

```bash
sudo add-apt-repository --remove ppa:PPA_Name/ppa
```

### Repo and packages

```bash
sudo apt-get install ppa-purge
sudo ppa-purge ppa-url
```
